# Icon Generation Report

## 繁體中文

- 已由 local script 產生 NeonDrift app icon master SVG、Unity branding icon、iOS icon、Android adaptive icon 與 WebGL favicon。
- Icon 風格遵守 spec review 的 icon contract：霓虹漂移、可讀輪廓、深色背景、青色/洋紅重點色，避免依賴平台專屬商標。
- 驗證條件：所有 PNG 必須可讀且符合指定尺寸；缺少任一平台 icon 時 development pipeline 不合格。

## English

- The local script generated the NeonDrift app icon master SVG, Unity branding icon, iOS icons, Android adaptive icons, and WebGL favicon.
- The icon style follows the spec review icon contract: neon drift, readable silhouette, dark background, cyan/magenta accent colors, and no platform-specific trademark dependency.
- Validation: every PNG must be readable and match the required size; missing any platform icon fails the development pipeline.
