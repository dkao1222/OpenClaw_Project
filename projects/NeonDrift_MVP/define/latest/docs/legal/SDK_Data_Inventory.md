# SDK Data Inventory

## 繁體中文

- Unity Ads: candidate SDK for rewarded ads and interstitial ads; requires data disclosure for Advertising Data, Device ID, Product Interaction, Diagnostics depending on configuration.
- Google AdMob: candidate SDK for Android and iOS ads; requires Google Play Data Safety and Apple privacy label updates when integrated.
- Unity Analytics: disabled by default for MVP unless explicitly approved.
- Firebase Analytics / Crashlytics: disabled by default for MVP unless explicitly approved.
- Attribution SDK: disabled by default; adding one requires ATT, privacy label, Data Safety, and privacy policy updates.
- data disclosure: every SDK must list collected data, shared data, purpose, linked/not linked status, tracking status, retention, and opt-out/consent behavior.
