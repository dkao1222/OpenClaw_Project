# App Store Privacy Label

## 繁體中文

- Data Used to Track You: 若啟用個人化廣告或跨 app/site tracking，需揭露 Device ID、Advertising Data、Product Interaction。
- Data Linked to You: 若 SDK 將 device identifier、ad identifier、purchase/account 或 gameplay profile 連到使用者，需標示 linked。
- Data Not Linked to You: crash logs、performance diagnostics、aggregate gameplay metrics 只有在去識別且不能重新連回使用者時才可標示 not linked。
- Apple App Tracking Transparency: 任何 IDFA 或跨 app/site tracking 都必須先取得 ATT permission。
- Third-party partners: Unity Ads、Google AdMob、Firebase、analytics 或 attribution SDK 都必須納入揭露。

## English

- Data Used to Track You: If personalized ads or cross-app/site tracking are enabled, disclose Device ID, Advertising Data, and Product Interaction.
- Data Linked to You: If an SDK links device identifiers, ad identifiers, purchase/account data, or gameplay profiles to a user, mark the data as linked.
- Data Not Linked to You: crash logs, performance diagnostics, and aggregate gameplay metrics may be marked not linked only when de-identified and not re-linkable.
- Apple App Tracking Transparency: any IDFA or cross-app/site tracking requires ATT permission first.
- Third-party partners: Unity Ads, Google AdMob, Firebase, analytics, or attribution SDKs must be included in the disclosure.
