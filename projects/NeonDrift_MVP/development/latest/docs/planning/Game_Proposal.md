# Game Proposal

## 繁體中文

### 遊戲目標
- NeonDrift MVP 提供一款 iOS-first 的短局霓虹漂移閃避遊戲，讓玩家在 30 到 90 秒內完成一輪清楚、有壓力、可立即重試的遊玩。
- 產品目標是先驗證核心操作、HUD 可讀性、失敗/重試節奏、60 FPS 目標與廣告變現前置條件，再擴展 Android 與 WebGL。

### 核心玩法
- 玩家以單手左右觸控控制車輛 drift，在霓虹賽道中閃避障礙、累積分數、維持 combo/boost 回饋。
- 每輪包含開始、行駛、風險升高、碰撞或失誤、結果顯示、立即重試；不加入帳號、多人或長期養成系統。

### 成功條件
- 新玩家能在首輪理解左右操作、分數、危險提示與 retry 入口。
- 本機 Unity build 可產出 iOS simulator app，QA 可驗證 Start、Settings、Best Score、Pause、Retry、safe area 與 frame pacing。

## English

### Game Goal
- NeonDrift MVP delivers an iOS-first short-session neon drift dodging game where one round is readable, tense, and immediately replayable within 30 to 90 seconds.
- The product goal is to validate core control feel, HUD readability, failure/retry pacing, 60 FPS target, and monetization readiness before expanding to Android and WebGL.

### Core Gameplay
- The player uses one-handed left/right touch steering to drift through a neon track, avoid hazards, score clean survival, and receive combo/boost feedback.
- Each run includes start, drive, rising risk, collision or mistake, result feedback, and retry; account, multiplayer, and long progression systems are outside MVP scope.

### Success Criteria
- A new player can understand steering, score, danger cues, and retry entry during the first run.
- Local Unity build can produce an iOS simulator app, and QA can verify Start, Settings, Best Score, Pause, Retry, safe area, and frame pacing.
