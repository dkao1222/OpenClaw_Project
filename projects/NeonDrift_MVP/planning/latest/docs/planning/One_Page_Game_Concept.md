# One Page Game Concept

## 繁體中文

### 產品願景
- NeonDrift MVP 是一款快速進入、快速失敗、快速重試的手機霓虹漂移遊戲，服務碎片時間中的 arcade 反應型玩家。

### 目標受眾
- 13 到 35 歲喜歡 runner、reaction、arcade racing 的手機玩家，重視低學習成本、單手操作、清楚 HUD 與穩定效能。

### 核心價值
- 單手左右 drift 立即回饋。
- 60 FPS 目標、safe area、分數、危險、pause/retry 都可被 QA 驗證。
- Rewarded ads 優先，避免過早破壞短局節奏。

### MVP 摘要
- MVP 包含主選單、遊戲 HUD、障礙生成、分數/combo、失敗結果、立即重試、基礎設定、隱私/廣告準備與本機 Unity build gate。

## English

### Product Vision
- NeonDrift MVP is a mobile neon drift game built for fast entry, fast failure, and fast retry for arcade reaction players in short breaks.

### Target Audience
- Mobile players age 13 to 35 who like runner, reaction, and arcade racing games, and value low learning cost, one-handed input, readable HUD, and stable performance.

### Core Value
- Immediate feedback from one-handed left/right drift control.
- QA-verifiable 60 FPS target, safe area, score, danger, pause, and retry contracts.
- Rewarded ads first, without interrupting the short-session loop too early.

### MVP Summary
- MVP includes main menu, run HUD, hazard spawning, score/combo, failure result, instant retry, basic settings, privacy/ads readiness, and local Unity build gate.
