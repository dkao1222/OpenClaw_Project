# Platform Direction

## 繁體中文

### 平台範圍:
- iOS平台為首發目標
- 支援iPhone 8及以上設備

### 控制方式:
- 單手操控設計，以螢幕左側滑動控制方向
- 精確的漂移控制與即時反饋

### 技術限制:
- 優化遊戲性能以適應較低配置設備
- 確保遊戲在不同螢幕尺寸上的一致性

## English

### Platform Scope:
- iOS platform as the primary launch target
- Support for iPhone 8 and above devices

### Control Model:
- One-handed control design with screen left-side swipe for direction
- Precise drift control with immediate feedback

### Technical Constraints:
- Optimize game performance for lower-end devices
- Ensure consistent gameplay across different screen sizes
