# Stage 03 Context Pack

## experience goal
- Deliver a fast, readable neon racing experience built around short runs and immediate retry.

## creative pillars
- Speed readability: hazards, rewards, and route cues stay readable at speed.
- Neon focus: visual energy supports play clarity.
- Instant retry confidence: failure feels fair and recoverable.

## player promise
- Every run feels quick, clear, and worth replaying immediately.
- Failure explains what the player can improve next.

## core input promise
- Steering, drift, boost, and restart respond immediately and predictably.

## operation flow scope
- MVP covers start, active run, scoring, failure, and fast retry.

## experience boundaries
- No multiplayer, accounts, long quests, complex progression, or UI layout ownership in Stage 03.
