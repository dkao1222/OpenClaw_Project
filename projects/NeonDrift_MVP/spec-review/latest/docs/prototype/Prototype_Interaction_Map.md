# Prototype Interaction Map

## 繁體中文

- Start: menu to gameplay start, visible transition and audio sting.
- Settings: menu to settings, toggles must be readable and reversible.
- Left / Right: player ship changes lane and trail direction within 80ms.
- Boost / Reward: pickup changes score, boost, combo, particles, and sound.
- Danger / Near Miss: warning and lane readability must make risk understandable.
- Collision / Failure: failure reason is visible before retry.
- Retry: returns to an early playable state with score/pulse/combo reset.
- Pause / Resume: freezes and restores current gameplay state.
