# HUD Spec

## 繁體中文

- HUD Spec 只定義 run 期間可見的 current score、combo/boost、speed/danger、pause 與 failure cue，不承擔 menu layout。
- HUD 必須遵守 safe area，避開 notch、Dynamic Island、home indicator，資訊優先順序為 survival/score > danger > boost/combo > pause。
- Acceptance: iPhone SE 2/3 與 iPhone 15 上所有 HUD 元件不遮擋車體、障礙、觸控區或視覺警示。

## English

- HUD Spec defines only visible in-run current score, combo/boost, speed/danger, pause, and failure cues; it does not own menu layout.
- HUD must follow safe area rules and avoid notch, Dynamic Island, and home indicator, with priority survival/score > danger > boost/combo > pause.
- Acceptance: on iPhone SE 2/3 and iPhone 15, HUD elements do not obscure the vehicle, hazards, touch zones, or warning cues.
