# Visual Audio Detail Spec

## 繁體中文

- 視覺以深色賽道、霓虹邊線、可讀障礙與低密度粒子為主；危險色不可和獎勵色混淆。
- Bloom 和 screen shake 必須保守使用，任何 VFX 不得遮住車體、障礙、HUD 或節奏提示。
- 音效需要 start、drift、near miss、score tick、collision、retry；haptics 僅用於碰撞與高價值回饋。

## English

- Visuals use a dark track, neon lane edges, readable hazards, and low-density particles; danger color must not conflict with reward color.
- Bloom and screen shake must be conservative, and VFX must not obscure the vehicle, hazards, HUD, or timing cues.
- Audio needs start, drift, near miss, score tick, collision, and retry cues; haptics are reserved for collision and high-value feedback.
