## 繁體中文

### 平台範圍:
- iOS平台為主要發布目標，確保遊戲在iPhone和iPad上都能提供流暢體驗。
- 支援iOS 15及以上版本，以利用最新的遊戲功能與效能優化。

### 控制方式:
- 單次觸控即可進行漂移操作，簡化控制邏輯以提升易用性。
- 畫面中顯示觸控提示，幫助玩家掌握最佳的漂移時機。

### 技術限制:
- 使用Unity 6000.0.71f1引擎進行開發，確保遊戲在iOS平台上穩定運行並支援未來升級。
- 遊戲效能需針對主流iOS設備進行優化，以維持流暢的遊戲體驗。

## English

### Platform Scope:
- iOS platform is the primary target for release, ensuring a smooth experience on both iPhone and iPad.
- Support iOS 15 and above to leverage the latest game features and performance optimizations.

### Control Model:
- Single touch input for drifting, simplifying control logic to enhance usability.
- On-screen touch indicators help players master the optimal timing for drifts.

### Technical Constraints:
- Development using Unity 6000.0.71f1 engine to ensure stable performance on iOS and support future upgrades.
- Game performance must be optimized for mainstream iOS devices to maintain a smooth gaming experience.
