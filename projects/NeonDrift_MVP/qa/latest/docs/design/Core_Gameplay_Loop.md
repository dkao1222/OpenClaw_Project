# Core Gameplay Loop

Document contract: this file records only the current stage decision surface for the next planning owner. Do not add sections, do not copy full prior documents, and keep each subsection to 1 bullet.

## 繁體中文

required keywords: core, gameplay, loop

### 範圍
- 核心循環由進入賽道、操控漂移、累積分數、承擔風險、失敗後立即重試構成。

### 決策
- 每一輪都要讓玩家在十秒內理解目標，並在三十秒內看到分數或技巧進步。

### 交接重點
- 成功條件聚焦於可讀速度感、穩定操控回饋，以及清楚的失敗原因。

## English

required keywords: core, gameplay, loop

### Scope
- The core loop starts with a run, asks the player to steer through risk, converts survival into score, and returns quickly to retry after failure.

### Decisions
- Each session should expose the objective within seconds and show measurable mastery within a short run.

### Handoff Notes
- Success depends on readable speed, dependable control response, and failure feedback that explains what to improve next.
